# Security Policy

## Supported Versions

Security fixes are provided for the latest minor version.

## Reporting a Vulnerability

Please do not open a public issue for security reports.

Use GitHub private vulnerability reporting:

https://github.com/SalmonPlays/oss-signal/security/advisories/new

Include:

- Affected version or commit
- Reproduction steps
- Impact
- Any suggested fix

Expect an initial response within 7 days. If the report is accepted, a fix and
advisory will be coordinated before public disclosure. If GitHub private
reporting is unavailable, use the contact route on the maintainer's public
GitHub profile without disclosing vulnerability details in a public issue.
